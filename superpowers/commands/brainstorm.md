## ⚠️ Deprecated Command

This command is deprecated and will be removed in the next major release.

### Migration

Use the **superpowers:brainstorming** skill instead:

```
Just say: "brainstorm this feature" or "let's brainstorm"
```

The skill auto-triggers when you mention "brainstorm", "头脑风暴", or "design" in conversation.

### Original Behavior

This command previously launched an interactive brainstorming session to explore user intent, requirements, and design before implementation.
