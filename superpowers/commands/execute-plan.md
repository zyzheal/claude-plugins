## ⚠️ Deprecated Command

This command is deprecated and will be removed in the next major release.

### Migration

Use the **superpowers:executing-plans** skill instead:

```
Just say: "execute the plan" or "run the implementation plan"
```

The skill auto-triggers when you mention "execute plan", "implementation", or "执行计划".

### Original Behavior

This command previously executed a written implementation plan with review checkpoints between task phases.
