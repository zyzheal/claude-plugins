# Code Simplifier Plugin

## 概述
专业代码简化 agent，在保持功能不变的前提下提升代码清晰度、一致性和可维护性。

## 安装
```
/plugin install code-simplifier@claude-plugins-official
```

## 激活
作为 agent 使用，在代码审查或重构任务中被其他 skill 调用。

## 使用方法
**触发方式：**
- 通过 code-review 流程自动调用
- 手动请求简化最近修改的代码

**简化策略：**
1. **保持功能不变** — 不改变代码行为，只改变写法
2. **应用项目标准** — 遵循 CLAUDE.md 中的编码规范
3. **提升清晰度** — 减少嵌套、消除冗余、改进命名

**适用场景：**
- 减少不必要的复杂度和嵌套
- 消除冗余代码和抽象
- 改进变量和函数命名
- 合并相关逻辑
- 移除明显代码的注释
- 避免嵌套三元运算符

## 参数
- `model: opus` — 默认使用 Opus 模型以获得最佳代码质量

## 依赖
- Claude Code 的 agent 系统
- 项目中的 CLAUDE.md 编码规范

## 示例
```
# 简化最近修改的代码
请简化这个 PR 中的代码变更

# 指定范围简化
请简化 src/utils/ 目录下的代码
```
